🧩 能力模組
核心只有靈魂＋記憶；其餘能力要才裝。所有模組都由靈魂來使用、都寫回同一片記憶迴廊。不常用的可以關掉。

📖 精讀

遇到看不懂的長句、術語？貼進來，夥伴會用你的程度帶你：先抓意義 → 圈骨架詞 → 講語感（不抠術語）→ 階梯造句。拆過的字都收進記憶。

🎭 場景模擬

想演一段情境？你出場景、定角色、給個小目標，夥伴就入戲陪你演，卡住隨時喊「暫停」跳出來問。

📝 複習題

迴廊裡到期該複習的星星，夥伴會挑出來出一份小卷（選擇＋填空，最多 12 題），當場批改、成績直接寫回記憶重新排程，做完還有夥伴講評。

📍 你的資料存在哪裡
對話、記憶迴廊、靈魂設定、程度紀錄，全部只存在【這台裝置的瀏覽器本機】。Noctilingo 沒有伺服器，不會把這些內容上傳到任何地方。清除瀏覽器資料會讓這些內容一併消失，建議定期用下面「💎 記憶結晶」匯出備份。

🔑 API Key 怎麼處理
你貼的 API key 只存在這台裝置的本機瀏覽器裡，直接送往你選擇的模型商（例如 Anthropic）進行對話。Noctilingo 不會、也無法看到或儲存你的 key，費用由你與模型商直接結算。匯出的「記憶結晶」永遠不包含任何 key。

🤖 關於 AI 內容
這裡的回覆由第三方語言模型生成，可能出現錯誤、過時或不準確的內容，僅供語言學習參考，不構成醫療、法律、心理諮商等專業建議。重要決定請自行查證或諮詢專業人士。

💛 一個提醒
Noctilingo 的靈魂是為了陪你練習語言而設計的 AI 角色，不是真人，也無法取代真實的人際關係或專業心理支持。如果你正經歷困難，也鼓勵找信任的人或專業資源聊聊。

🔞 適用年齡
本產品內容以一般成人與青少年學習情境設計，未成年使用者建議在家長或監護人了解下使用。
